1. **Open the game** on your phone or device
2. **Roam the room** вҖ” talk to people and find who matches each square
3. **Tap to mark** a square when you find a match
4. **Get 5 in a row** (any row, column, or diagonal) to win рҹҺү

Each game is **randomized** вҖ” no two boards are the same!

---

## вңЁ Features

- рҹҺІ **Randomized boards** вҖ” shuffled fresh each game with Fisher-Yates
- рҹ’ҫ **Auto-saves** вҖ” resume mid-game after closing the tab
- рҹ“ұ **Mobile-friendly** вҖ” designed for phones at events
- вҡЎ **Instant deploy** вҖ” push to `main` and it's live on GitHub Pages
- рҹҺЁ **Fully customizable** вҖ” swap questions to match any event theme

## рҹҡҖ Getting Started

### Prerequisites

- [Node.js 22](https://nodejs.org/) or higher

### Run locally

```bash
npm install
npm run dev
```

### Build for production

```bash
npm run build
```

Deploys automatically to GitHub Pages on push to `main`.

## рҹӣ  Customize for Your Event

**Change the questions** вҖ” edit [`src/data/questions.ts`](src/data/questions.ts) with 24 prompts that fit your crowd:

```ts
export const questions: string[] = [
  "has shipped a feature this week",
  "uses a mechanical keyboard",
  "has attended a hackathon",
  // ... 21 more
];
```

Ideas: team facts, conference themes, hobbies, local trivia вҖ” anything that sparks conversation!
